# 📥 YouTube Media Downloader

A high-performance Python utility for downloading video and audio content from YouTube. This project showcases the ability to integrate external libraries, manage file systems, and handle media streams efficiently.

## 🚀 Features
* **Format Flexibility:** Supports downloading full video (MP4) or high-quality audio extraction (MP3).
* **Stream Optimization:** Automatically identifies and selects the highest available resolution.
* **Progress Feedback:** Provides the user with real-time status updates during the download process.
* **Sanitized File Naming:** Automatically cleans titles to ensure compatibility across all operating systems.

## 🧰 Tech Stack
* **Language:** Python 3.x
* **Core Dependency:** `yt-dlp` (The industry-standard for media extraction).
* **File I/O:** Automated directory management for organized downloads.
